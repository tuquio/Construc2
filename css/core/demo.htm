The heading classes .H1 … .H6

More often than not you want a piece of (short) text to look like a specific heading level of your theme including it's font and color settings but without using the sematics associated with that heading element. The .H1 to .H6 classes complement the styles assigned to the true heading elements for exactly that visual purpose. If you make it your habit to always add any of the heading classes to your heading element rules this not only saves you a ton of CSS rules, it also helps to make the overall look of your theme to be more consistent and easier to manage.

You can then have the semantically corrent <H3> element in your markup but make it look like an <H4> simply by setting the .H4 class and add this rule to your selectors: 

.mod :any(h3, h4, .H3, .H4) {font-size:140%;color:navy}
